Install iliterate (desktop)

This is the temporary install flow while you’re in private beta.

  1. Download the iliterate folder (your beta zip / shared folder).
  2. Open Chrome → go to chrome://extensions.
  3. Turn on Developer mode (top right).
  4. Click Load unpacked → select the extension folder.
  5. Pin iliterate (puzzle piece → pin).
  6. Highlight text → right-click → Go iliterate.

When you launch to the Chrome Web Store, this page becomes one button.
