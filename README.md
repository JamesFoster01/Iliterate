# Iliterate — AI Study Assistant

Iiterate is an AI-powered tool designed to help students summarise learning material and focus on key concepts efficiently.

## Features
- AI text summarisation
- Study optimisation workflow
- Browser extension integration
- Backend API server

## Tech Stack
- JavaScript
- Node.js
- HTML/CSS
- API integration

## Structure
landing/ → landing page  
extension/ → browser extension  
server/ → backend API  

## Purpose
Built as a personal project to explore real-world AI integration and automation workflows.
